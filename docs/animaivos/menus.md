# menus

Módulo criado para guardar os menus de sua animação.

Encontra-se no arquivo `animaivos/menus.py`.

::: animaivos.menus
    handler: python
    show_source: true

       1. Cena 1
       2. Cena 2
       3. Cena 3
       4. Cena 4
       5. Sair
       
    Escolha sua opção: """
